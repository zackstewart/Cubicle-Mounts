## Required Materials

### Base-Hooks:
  - 3-Hook Compression Mount   
    - 1x M3x5mmx4mm 



### Mounts:
  - 3-Hook DIN Rail Mount
    - 1x M5x6mmx7mm
    - 1x M5 Screw

  - 3-Hook Long Power Bar Snap-In Mount
    - None

  - 3-Hook Slide On Hanger
    - None
